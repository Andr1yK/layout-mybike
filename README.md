## Description

MyBike is the landing page of the Ukrainian motorcycle shop MyBike

## Technologies
- Adaptive layout
- HTML5
- SCSS
- BEM
- JS

## Local development

### Dependencies
* Node v14.18.2 and higher
* NPM v8.5.5 and higher

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
